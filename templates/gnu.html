GNU/Linux on Aakash

Aakash can run Android as well as GNU/Linux. GNU/Linux can be burned on the external SD card and will not effect existing Android system. It has Debian package manager and can run all GNU application with almost no modification. Distros such as Debian, Linaro, Ubuntu & Fedora have been tested to run on Aakash. Please visit the documentation on 'How to port GNU/Linux on Aakash'. Also visit the publications section for more detail.

expEYES

expEyes is a tool to learn science by exploring and experimenting. It can be run on Aakash to carry out 50 documented experiments. It is a hardware & software framework for developing science experiments, demonstrations and projects without getting in to the details of electronics or computer programming. It converts Aakash into a science laboratory.

Oscad

Oscad is an open source EDA tool for circuit design, simulation, analysis and PCB design. It is an integrated tool built using open source software such as KiCad, Ngspice and Scilab. The image shows Oscad(Ngspice simulation) running on Aakash.

Anuduino

It is a two dollar arduino compatible usb development board based on ATtiny-85 micro-controller developed at Indian Institute of Technology Bombay. Adjacent image shows its usage to measure room temperature. Visit documentation on how to build it from scratch.
